# Y-UI 组件库

Y-UI 是一个轻量级的 React 组件库，提供了一系列简单易用、可定制的 UI 组件，帮助开发者快速构建现代化的 Web 应用。

## 特点

- **简单易用**：API 设计简洁明了，易于上手
- **可定制**：支持自定义样式和主题
- **TypeScript 支持**：使用 TypeScript 开发，提供完整的类型定义
- **模块化**：可按需引入组件，减小打包体积

## 安装

```bash
# 使用 npm
npm install @rainliu/y-ui

# 使用 yarn
yarn add @rainliu/y-ui

# 使用 pnpm
pnpm add @rainliu/y-ui
```

## 快速开始

```jsx
import React from "react";
import { Button } from "@rainliu/y-ui";

function App() {
  return (
    <div>
      <Button type="primary">点击我</Button>
    </div>
  );
}

export default App;
```
