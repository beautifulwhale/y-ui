# Calendar

Calendar 组件

## 安装

```bash
pnpm add @y-ui/calendar
```

## 使用

```jsx
import { Calendar } from '@y-ui/calendar';

function App() {
  return (
    <Calendar>
      内容
    </Calendar>
  );
}
```

## API

| 属性 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 子元素 | ReactNode | - |
| className | 自定义类名 | string | - |
