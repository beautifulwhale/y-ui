# Message

Message 组件

## 安装

```bash
pnpm add @y-ui/message
```

## 使用

```jsx
import { Message } from '@y-ui/message';

function App() {
  return (
    <Message>
      内容
    </Message>
  );
}
```

## API

| 属性 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 子元素 | ReactNode | - |
| className | 自定义类名 | string | - |
